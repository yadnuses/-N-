# Tech Stack

## 运行环境
- 纯静态网页，无需后端服务器
- 目标浏览器：现代桌面浏览器（Chrome / Firefox / Safari / Edge）

## 前端技术
- **HTML5** — 页面结构与语义化标签
- **CSS3** — 自定义样式、动画、响应式布局
- **原生 JavaScript (ES5/ES6)** — 无框架、无构建步骤
- **localStorage** — 本地存档与进度持久化

## 部署方式
- 可直接通过 `file://` 打开，或部署到任意静态托管（GitHub Pages / Netlify / Vercel 等）

## 不使用的技术
- 无前端框架（React / Vue / Angular）
- 无构建工具（Webpack / Vite / Parcel）
- 无 CSS 框架（Bootstrap / Tailwind）
- 无后端服务、数据库或用户账号系统
